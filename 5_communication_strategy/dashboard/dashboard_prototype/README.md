## Caching Strategy
- Use Streamlit caching for metrics and model resources to reduce load times and repeated computation.

Example:
```python
@st.cache_data(ttl=3600)
def load_metrics():
    # load and return metrics
    pass

@st.cache_resource
def load_model():
    # load and return ML model
    pass
```

## Lazy Loading
- Metrics load immediately on app start.
- Large model is loaded only when prediction functionality is invoked.
- Predictions and large datasets are streamed/loaded in chunks where possible.
